Transparenz & Sicherheit

Kassenwart Pro ist quelloffen verteilt und prüfbar. Hier kann die Integrität der heruntergeladenen Datei überprüft und eine Windows-Sicherheitswarnung eingeordnet werden.

🔒 Datei-Integrität prüfen (SHA256)

Jede Kassenwart-Pro-Datei hat eine eindeutige Prüfsumme (SHA256-Hash). Damit lässt sich sicherstellen, dass die heruntergeladene Datei exakt der veröffentlichten Version entspricht — unverändert und vollständig.

SHA256 · KassenwartPro.exe (v1.0)
092db793cb1e39e071536a41b242bb72b650feb2a631464c61203ab8eafab9d5

So wird die Prüfsumme überprüft:

1
PowerShell öffnen Windows-Taste drücken, PowerShell eintippen, Enter.
2
Prüfsumme berechnen Folgenden Befehl eingeben (Pfad ggf. anpassen):
Get-FileHash "C:\Users\IhrName\KassenwartPro\KassenwartPro.exe" -Algorithm SHA256
3
Ergebnis vergleichen Die angezeigte Hash-Zeichenkette muss exakt mit dem oben stehenden Wert übereinstimmen. Stimmen beide überein, ist die Datei unverändert.

🛡️ Windows SmartScreen — Warnung einordnen

Beim ersten Start von Kassenwart Pro kann Windows eine Sicherheitswarnung anzeigen: „Der Computer wurde durch Windows geschützt" oder „Windows Defender SmartScreen hat eine unbekannte App blockiert."

ℹ️

Das ist kein Virus. Microsoft SmartScreen warnt bei jeder neuen Software, die noch nicht häufig genug heruntergeladen wurde, um einen „Ruf" aufzubauen. Das betrifft praktisch alle kleinen, unabhängigen Programme ohne teures Code-Signing-Zertifikat — unabhängig davon, ob sie sicher sind oder nicht.

So wird die App trotzdem gestartet:

1
„Weitere Informationen" klicken In der SmartScreen-Warnung auf den kleinen Link „Weitere Informationen" klicken.
2
„Trotzdem ausführen" wählen Es erscheint ein Button „Trotzdem ausführen". Damit startet die App ganz normal.

Die Integrität der Datei kann jederzeit mit der oben stehenden SHA256-Prüfsumme verifiziert werden.

Dabei helfen, die Warnung dauerhaft zu entfernen:

Je mehr Nutzer die Datei bei Microsoft als sicher melden, desto schneller verschwindet die SmartScreen-Warnung für alle. Eine solche Meldung dauert weniger als eine Minute und hilft, Kassenwart Pro als vertrauenswürdig einzustufen.

1
Microsoft-Meldeportal öffnen Die folgende Seite aufrufen (Microsoft-Konto erforderlich):
→ Microsoft Security Intelligence — Datei einreichen
2
Datei hochladen Als Kategorie „Software that should not be detected (False Positive)" wählen und die Datei KassenwartPro.exe hochladen.
3
Absenden Formular absenden. Microsoft prüft die Datei und passt die Einstufung ggf. an.

Alternativ kann die Meldung auch direkt aus Microsoft Edge erfolgen: Beim Download-Hinweis auf „Diese Datei als sicher melden" klicken und das Formular ausfüllen.
